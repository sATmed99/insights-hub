## 3-Step Deployment (5 minutes total)

### Step 1: Push to GitHub
```bash
cd insights-hub
git add .
git commit -m "Deploy: Global Startup Analytics Hub ready for Streamlit Cloud"
git push origin main
```

### Step 2: Deploy to Streamlit Cloud
1. Go to: **https://streamlit.io/cloud**
2. Click: "Create app"
3. Select:
   - Repository: Your repo (e.g., `German-University`)
   - Branch: `main`
   - File: `streamlit_app.py`
4. Click: "Deploy" (wait 2-3 minutes)

### Step 3: Get Your Public URL ✅
You'll receive: `https://insights-hub-[username].streamlit.app/`

4. Check data files are committed to GitHub
